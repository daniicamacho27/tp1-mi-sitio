# Comando usados 
## 1.pwd
Muestra la carpeta actual en la que estoy trabajando.
Lo use para verificar en que carpeta estaba.
## 2.ls
Lista los archivos dentro de una carpeta.
Lo use para ver index.html y style.css.
## 3.cd
Permite entrar a una carpeta.
Lo use para entrar a tp1-mi-sitio
## 4.mkdir
Crea una carpeta nueva.
Lo use para crear la carpeta tp1-mi-sitio.
## 5.touch
Crea un archivo vacío.
Lo use para crear index.html y style.css.
## 6.cat
Muestra el contenido de un archivo.
Lo use para ver el contenido de algunos archivos.
## 7.which
Muestra dónde esta instalado un programa.
Lo use para verificar donde está Git.
## 8.code
Abre Visual Studio Code desde la terminal.
Lo use para abrir archivos.
## 9.git --version
Muestra la version de Git instalada.
Lo use para comprobar y verificar que version de Git tenia.
## 10. git config --list
Muestra la configuracion de Git con usuario y email.
Lo use para configuar Git.
